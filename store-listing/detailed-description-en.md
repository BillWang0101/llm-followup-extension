📌 Reading a long LLM answer and want to ask a follow-up about a term? You shouldn't have to scroll to the input box, ask, and scroll back. This extension fixes exactly that.

✨ **Select → ask → answer appears inline**
Highlight any text in the model's reply and a follow-up bubble pops up right next to it. Type your question, hit Ctrl+Enter, get a streaming answer in place. Your eyes never leave the paragraph you were reading.

🎯 **Two modes, your choice**
• Answer in bubble: calls your own API key, answer streams inside the bubble, main chat stays untouched.
• Inject into main chat: drops the follow-up into the page's chat input and sends it—reuses your logged-in session, zero extra cost.

💬 **Multi-turn inside the bubble**
A persistent input at the bottom lets you keep drilling deeper. Select text inside a bubble answer and a "Ask about this" chip appears—truly recursive read-as-you-ask.

🌐 **10 providers, one extension**
OpenAI (GPT-5.4 family), Anthropic (Claude 4.6), Google Gemini, DeepSeek, Moonshot Kimi, Zhipu GLM-5.1, MiniMax M2.7, ByteDance Doubao, Alibaba Qwen, SiliconFlow. Use whichever you already pay for.

🔒 **Privacy-first**
• API Key stored only in chrome.storage on your device
• Requests go straight to your chosen provider's official endpoint—never through any third party
• No telemetry, no tracking, no ads
• API domains use runtime permission requests: only the providers you actually use get access

🎨 **Polished UI**
Dark-first design inspired by Linear. Inter Variable font, semi-transparent borders, system-theme aware. 140ms spring animation on appearance—never jarring.

📖 Supported sites: ChatGPT, Claude, Gemini, DeepSeek, Kimi, Doubao, ERNIE, Qwen Chat

🔑 Setup: toolbar icon → Settings → pick provider + paste key → done
