📌 阅读大模型的长回答，遇到不懂的概念要追问——却发现得滚到底部输入框、问完再翻回原段落？这个扩展就是来根治这种打断的。

✨ **划词即问，答案就地浮现**
选中任意一段回答里的文字，旁边立刻浮出一个追问气泡，输入问题按 Ctrl+Enter 就能得到流式回答。视线不动，节奏不断。

🎯 **两种模式任选**
• 气泡内解答：调用你自己的 API Key，答案在气泡里流式返回，不打扰主对话。
• 注入主对话：把追问直接填到页面输入框并发送，复用你已登录的会话，零额外成本。

💬 **气泡内多轮对话**
答案下方常驻输入框，可以一路追问下去。在气泡里的答案里再划词，还能弹出"追问这段"，实现真正的边读边问、层层深入。

🌐 **覆盖主流 9 家大模型**
OpenAI（GPT-5.4 系列）、Anthropic（Claude 4.6）、Google Gemini、DeepSeek、月之暗面 Kimi、智谱 GLM-5.1、MiniMax M2.7、字节豆包、阿里通义千问——有一家就能用。

🔒 **隐私优先**
• API Key 仅保存在你本地浏览器的 chrome.storage
• 请求只发给你选定的 Provider 官方端点，不经过任何第三方服务器
• 无遥测、无埋点、无广告
• API 域走"按需授权"：你用到哪家才请求哪家的访问权

深色优先的 Linear 风格 UI，Inter 字体，半透明边框，跟随系统主题。气泡出现有 140ms 弹性动画，不会突兀。
